Contour Detection and Grouping


A preliminary release of my fractional programming grouping algorithm is available here: fpg.zip

The parameters are not tuned for the gPb detector; a full-blown release, including the subsequent works will be uploaded by July.

The code is in Matlab, but you will need to install mosek (they provide a free and infinitely renewable academic license).

If you use this code in your research, please cite:

I. Kokkinos,
Highly-Accurate Boundary Detection and Grouping
Proc. IEEE Conf. on Computer Vision and Pattern Recognition (CVPR), 2010. [pdf]
